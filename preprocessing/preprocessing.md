# Dataset Preprocessing and Exploration

This directory contains utilities for inspecting, analyzing, and visualizing the YOLOv8 marine debris dataset prior to model training.

The preprocessing step is intended for exploratory analysis only and does not modify the dataset.

---

## üéØ Purpose

The preprocessing module provides tools to:

- Load class definitions from the YOLO `data.yaml` file
- Count object instances per class and dataset split
- Visualize class distribution across training, validation, and test sets
- Display example images with YOLO bounding box annotations

This step is useful for understanding class imbalance, dataset composition, and annotation quality.

---

## üß† Main Component

### `PreProcessorYoloV8`

A utility class designed to work with datasets following the standard YOLOv8 directory structure.

Main functionalities include:

- **Class loading**  
  Reads class names directly from `data.yaml`.

- **Object counting**  
  Counts the number of labeled objects per class for each dataset split (`train`, `valid`, `test`).

- **Class distribution visualization**  
  Generates grouped bar plots showing object counts and relative percentages per split.

- **Qualitative inspection**  
  Displays randomly selected example images per class with bounding boxes drawn from YOLO annotations.
